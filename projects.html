Representative Work

Small, focused systems that unblock real teams.

Below are examples chosen to show the kind of work ProVision Systems takes on: practical software and integration projects that clean up data, reduce manual work, and create real visibility for operations.

Example 1

Access control muster dashboard for life-safety.

Built a live “who is on site” dashboard backed by an enterprise access control system. The system ingests automatically exported reports every few minutes and updates a simple, high-clarity dashboard used by safety personnel during drills and incidents.

  • Automated parsing of CSV/XLS muster reports from vendor system
  • Normalized data model for people, badges, and entry status
  • Web UI showing current “IN” status and last-seen information
  • Designed so the customer can adjust rules and views over time

Example 2

Renovation project BOM + parts tracking.

For a multi-phase renovation effort, built a structured way to track parts across drawings, BOMs, and orders. The goal was not a giant ERP replacement, but a small system that made it obvious what was ordered, received, staged, and installed.

  • Ingested and cleaned existing BOM spreadsheets and PDFs
  • Created a single source of truth keyed by drawing & device
  • Simple status workflow (Planned → Ordered → Received → Installed)
  • Exportable views for PMs, field teams, and accounting

Example 3

Operations-focused AI assistants (in progress).

Prototyping narrow, operations-focused AI assistants that sit on top of existing documents, SOPs, and tickets — not general chat tools. The emphasis is on routing, summarization, and suggested responses that a human quickly reviews and sends.

  • Document ingestion from PDFs, Word, and existing wikis
  • Search and retrieval tuned for specific roles and scenarios
  • “Suggested reply” and “first draft” workflows for support
  • Guardrails and logging designed for regulated environments
